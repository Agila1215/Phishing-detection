# Phishing Detection using Machine Learning

## 📌 Description
This project detects phishing URLs using Machine Learning techniques. 
It scans QR codes, extracts URLs, and classifies them as Safe, Suspicious, or Malicious.

## 🛠️ Technologies Used
- Python
- Machine Learning
- Streamlit
- SVM (Support Vector Machine)
- Random Forest
- Ensemble Method

## 🚀 Features
- QR Code scanning
- URL extraction
- Real-time phishing detection
- Ensemble-based classification

## ▶️ How to Run
1. Install required libraries
2. Run the Streamlit app:
   streamlit run app.py
3. Open in browser
